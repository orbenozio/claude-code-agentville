# Agentville 🏘️

ויזואליזציית דסקטופ חמודה ופשוטה (top-down) של מה שהסוכנים וסוכני המשנה של Claude Code עושים *כרגע*.

## הרעיון בקצרה

עיירה קטנה שבה "גרים" הסוכנים שלך. כל סוכן הוא דמות קטנה:

- **idle** — נח בבית / במיטה.
- **עובד** — נכנס ל"מפעל" ומציג מעליו בועת דיבור עם המשימה שעליה הוא עובד.

המטרה היא לתת **נראות ויזואלית** למה שקורה בשיחת Claude Code — כמה סוכנים רצים במקביל, מה כל אחד עושה, ומתי הם מתחילים/מסיימים — בלי להשקיע בגרפיקה כבדה. ספרייטים פשוטים (אפילו ריבועים צבעוניים עם שם) מספיקים ל-MVP, ואפשר לשדרג בהמשך.

## מצבים שכדאי להציג

מעבר ל-idle / עובד, דווקא הרגעים האלה הכי שווים ויזואלית:

- ⏳ ממתין לאישור הרשאה
- ⚠️ נתקע / שגיאה
- 😴 ממתין ל-rate limit
- ✅ סיים → חוזר הביתה לנוח

## מאיפה מגיע המידע (אבן הבסיס לבדיקה)

Claude Code כותב את מהלך השיחה כקבצי JSONL תחת:

```
~/.claude/projects/<path-מקודד>/*.jsonl
```

קריאות ל-Task (סוכני משנה) אמורות להופיע שם גם כן. כלומר אפשר **לעקוב (tail) אחרי הקבצים** ולגזור מהם feed בזמן אמת — בלי אינטגרציה רשמית.

**הצעד הראשון בפרויקט:** לאמת בפועל מה באמת יש בקבצי ה-JSONL ואיך מזהים מתוכם סוכן/סוכן-משנה, מצב, ומשימה.

## שאלת ארכיטקטורה פתוחה

האם הכלי הוא **צפייה בלבד** (רק מציג), או גם **פעולה** (לעצור סוכן, לאשר הרשאה, לקפוץ לשיחה)? משפיע על הארכיטקטורה כבר מההתחלה.

## מועמדים טכנולוגיים

- **Tauri** (בינארי קטן וקליל) או **Electron** — לקונטיינר הדסקטופ.
- **Canvas / Pixi.js** — לאנימציה של הדמויות.
